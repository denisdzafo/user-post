## Installation

Install dependencies

    composer install
    
    npm install

Setup ENV file

    create database, and setup ENV file for database migrations

    
Application key generate

    php artisan key:generate

Run migrations    
    
    php artisan migrate
    
    
Generate Jwt key
    
    php artisan jwt:secret

Database seed

    php artisan db:seed

Project Setup

    Laravel 9, Vue.js 3, MySql, JWT Auth
